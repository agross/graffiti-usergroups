Hi!

Thank you for installing Graffiti-UserGroups.

This site helps you to set up Graffiti-UserGroups on your site. We recommend trying out Graffiti-UserGroups on a new and empty Graffiti site. We will set up everything you need so you can play around with the user group plugins and macros.

  • #set($action = "create-event-category") $macros.LoadThemeView("demosite/ajax-status.view") Create category for events.
    The category will be named "$demoSite.EventCategoryName()".
  • #set($action = "configure-event-plugin") $macros.LoadThemeView("demosite/ajax-status.view") Configure Event plugin.
    We will set up all plugin properties such that the theme works.
  • #set($action = "enable-event-plugin") $macros.LoadThemeView("demosite/ajax-status.view") Enable Event plugin.
    Enabling the plugin is a mandatory step to make the event macros work.
  • #set($action = "create-registration-post") $macros.LoadThemeView("demosite/ajax-status.view") Create registration post.
    We will create post called "Register" to enable registration for your events.
  • #set($action = "create-sample-events") $macros.LoadThemeView("demosite/ajax-status.view") Create sample events.
    We will create a couple of sample events so you can play around.
  • #set($action = "create-talk-category") $macros.LoadThemeView("demosite/ajax-status.view") Create category for events.
    The category will be named "$demoSite.TalkCategoryName()".
  • #set($action = "configure-talk-plugin") $macros.LoadThemeView("demosite/ajax-status.view") Configure Talk plugin.
    We will set up all plugin properties such that the theme works.
  • #set($action = "enable-talk-plugin") $macros.LoadThemeView("demosite/ajax-status.view") Enable Talk plugin.
    Enabling the plugin is a mandatory step to make the talk macros work.
  • #set($action = "create-sample-talks") $macros.LoadThemeView("demosite/ajax-status.view") Create sample talks.
    We will create a couple of sample talks so you can play around.
  • #set($action = "create-navigation-links") $macros.LoadThemeView("demosite/ajax-status.view") Create navigation links.
    We will add navigation links (to be found on top of the home page) so you can navigate through events and talks and also register for events.

Your turn!

Read the documentation or browse the source code to become familiar with what Graffiti-UserGroups provides. Then, create or customize your own theme.

You can now rename "home.layout.setupcomplete.view" to "home.layout.view" to disable this setup page. If you rename the category of events and talks, be sure to also rename the according view files "events.view", "events.post.view", "talks.view" and "talks.post.view", respectively.
