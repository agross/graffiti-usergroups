Hi!

Thank you for downloading and installing Graffiti-UserGroups.

This site helps you to set up Graffiti-UserGroups for your site. It's a good way to try out Graffiti-UserGroups on a new and empty Graffiti site. We will set up everything you need so you can play around with the user group plugins and macros.

  • #set($action = "create-event-category") $macros.LoadThemeView("demosite/ajax-status.view") Create category for events.
    The category will be named "Events".
  • #set($action = "configure-event-plugin") $macros.LoadThemeView("demosite/ajax-status.view") Configure Event plugin.
    We will set up all plugin properties such that the theme works.
  • #set($action = "enable-event-plugin") $macros.LoadThemeView("demosite/ajax-status.view") Enable Event plugin.
    Enabling the plugin is a mandatory step to make the macros work.
  • #set($action = "create-registration-post") $macros.LoadThemeView("demosite/ajax-status.view") Create registration post.
    We will create post called "Register" to enable registration for your events.
  • #set($action = "create-sample-events") $macros.LoadThemeView("demosite/ajax-status.view") Create sample events.
    We will create a couple of sample posts so you can play around.
  • Your turn!
    Read the documentation or browse the source code to become familiar with what Graffiti-UserGroups provides. Then, create or customize your own theme.
